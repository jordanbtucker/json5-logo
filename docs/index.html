JSON5 Logo

This repository contains SVG logos and icons for JSON5.

json5-logo-light.svg, json5-icon-light.svg

For use on light backgrounds. It displays text in near black and gray.

json5-logo-dark.svg, json5-icon-dark.svg

For use on dark backgrounds. It displays text in near white and gray.

json5-logo-auto.svg, json5-icon-auto.svg

For use on media that supports prefers-color-scheme. It displays text in near white and gray when the user has notified the operating system or user agent that they prefer a dark theme; otherwise, it displays text in near black and gray. Change the operating system or user agent theme to see how it adjusts.
